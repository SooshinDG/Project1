# PR #4: /pricing 페이지 외주 전환형 정리

> **브랜치:** `feature/pricing-conversion`  
> **범위:** app/pricing/page.tsx 및 pricing 관련 컴포넌트만. 다른 페이지/섹션 미변경.

---

## 변경 요약

- **3단 패키지 카드 (Lite / Standard / Premium)**  
  - 각 카드에 "포함" 핵심 4~6개만 짧게 정리(랜딩 1페이지, 모바일 최적화, SEO, 문의 폼, 수정·납기 등).
- **Standard**  
  - 배지 문구 "가장 인기" → **"가장 많이 선택"** 으로 변경.
- **CTA**  
  - Primary: **"무료 진단 받기(3분)"** → /contact (전 카드 동일).  
  - Secondary: **"패키지 문의"** 링크 → /contact (선택적 유지).
- **미포함(별도) 섹션**  
  - 문구작성(카피), 촬영, 로고·브랜드, 도메인·호스팅, 추가 페이지, 추가 수정 비용을 명시.
- **비교표**  
  - 항목 정리: 납기, 수정 횟수, 페이지 수, 분석 도구, SEO, 유지보수 옵션.  
  - 데스크톱 테이블 + 모바일 카드 유지.
- **히어로**  
  - 라벨 "Pricing" → "패키지", 설명에 "무료 진단으로 시작" 문구 추가.
- **톤**  
  - A안 프리미엄 미니멀 유지(전역 토큰만 사용).

---

## 테스트 방법

1. `npm run dev` 실행 후 `http://localhost:3000/pricing` 접속.
2. 3단 카드·비교표·미포함(별도) 섹션 노출 확인.
3. "무료 진단 받기(3분)", "패키지 문의" 클릭 시 /contact 이동 확인.
4. (선택) 스크린샷: 패키지 카드·비교표·미포함 섹션 캡처 후 PR에 첨부.
