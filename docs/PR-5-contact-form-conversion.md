# PR #5: /contact 무료 진단 폼 전환형 정리

> **브랜치:** `feature/contact-form-conversion`  
> **범위:** app/contact/page.tsx 및 components/contact/contact-form.tsx 만. 다른 페이지 미변경.

---

## 변경 요약

- **폼 상단**  
  - "제출 후 절차(다음 단계)" 3줄 안내 추가: (1) 신청서 검토 후 24시간 이내 연락 (2) 맞춤 진단·견적 안내 (3) 무료 상담(원하실 경우).
- **필드 구성**
  - **[필수]** 이름/회사명(상호) — 단일 입력.
  - **[필수]** 연락처 — 이메일 또는 전화번호 중 1개만 입력해도 됨(유효성 검사).
  - **[필수]** 업종 — 칩 선택(카페·음식, 뷰티·헬스, 교육, 의료, 소매·이커머스, 서비스, 기타). 기타 선택 시 텍스트 입력.
  - **[필수]** 목표 — 예약 / 문의 / 구매 / 소개 중 선택.
  - **[선택]** 현재 사이트 URL, 예산 범위(선택지), 희망 일정(선택지), 참고 링크/메모(텍스트).
- **유효성 검사**  
  - 제출 시 필수 항목만 검사(이름·회사명, 연락처 1개 이상, 업종, 목표). 오류 시 해당 필드 아래 메시지 표시.
- **제출 후 성공 UI**  
  - "접수가 완료되었습니다", "영업일 기준 24시간 이내에 연락드리겠습니다." + **패키지 보기** 버튼 → /pricing 링크.
- **톤**  
  - A안 유지(전역 토큰만 사용).

---

## 테스트 방법

1. `npm run dev` 후 `http://localhost:3000/contact` 접속.
2. 필수만 입력 후 제출 → 성공 화면 및 "패키지 보기" → /pricing 이동 확인.
3. 필수 누락 시 제출 → 해당 필드 아래 오류 메시지 확인.
4. 연락처: 이메일만 입력 또는 전화만 입력 시 제출 가능한지 확인.
